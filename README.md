# GenBank Data — Bioinformatics & Text Algorithms

A full-stack application for scraping, storing, and analyzing DNA sequences from NCBI GenBank, built as a group project for the **Text Algorithms and Bioinformatics course at Université Paris Dauphine – PSL**.

scraping pipeline, backend API, text algorithm implementations, and frontend integration.

## Features

- **GenBank scraping** — automated retrieval of DNA sequence records from NCBI
- **Text algorithms for DNA** — pattern matching, sequence alignment, and string search on biological sequences
- **Full-stack interface** — browse, search, and analyze stored sequences through a web UI

## Structure

- `src/` — core application code (scraping, algorithms, API, and frontend)
- `requirements.txt` — Python dependencies
